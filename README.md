# 2048 Game

A modern implementation of the popular 2048 puzzle game built with vanilla JavaScript, HTML, and SCSS.

## Overview

2048 is a sliding block puzzle game. The objective is to slide numbered tiles on a grid to combine them and create a tile with the number 2048. It's a game of strategy and quick thinking that's fun and addictive!

## Features

- **Responsive Design** - Works seamlessly on desktop and mobile devices
- **Smooth Animations** - Fluid tile movements and merge effects
- **Score Tracking** - Keep track of your best score
- **Game Controls** - Keyboard and touch gesture support
- **Clean UI** - Modern, intuitive interface

## How to Play

1. Use arrow keys (or swipe on mobile) to move tiles
2. Tiles with the same number merge when they touch
3. After each move, a new tile appears randomly
4. Keep combining tiles until you reach 2048
5. Try to get the highest score possible!

## Technologies Used

- **HTML5** - Markup structure
- **SCSS** - Styling and responsive design
- **JavaScript (ES6)** - Game logic and interactivity

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm start
```
